En este repositorio encontrarán 2 carpetas

  • BackEnd
  • FrontEnd

BackEnd es una API construida en JAVA usando el Framework Spring Boot (sobre el IDE IntelliJ) y Hybernate para conectar a la MySQL Database

Es un ejemplo sencillo de una sola tabla con el modelo de un Student (name, email, dateofbirth and age)


FrontkEnd es un html (usando el framework Bootstrap y Vanilla JavaScript

Este front consume la API desarrallada en el BackEnd, haciendo Fetch a los endpoints.

El fetch se hace usando Vanilla JS (con async, await, try, catch), pero se podría haber usado jQuery con Ajax


Para probar el proyecto, deben descargar el Back, abrirelo y correrlo con IntelliJ

En application.properties deben configurar su Database: host/schema, username, password
